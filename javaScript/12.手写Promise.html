# 手写 Promise

# 确定 Promise 的大体结构

在我们手写 Promise 之前，首先我们要回顾一下，系统的 Promise 有哪些特征？

  1. 它有三种固定的状态，默认是pending。

  2. 它可以使用new关键词创建实例。

  3. 它需要传入一个函数，函数中可以调用resolve以及reject两个函数，调用resolve的时候状态会更改为fulfilled，调用reject的时候状态会更改为rejected。

  4. Promise的状态是唯一的，一旦发生改变是不可逆的，因此只有在状态为pending的时候才可以更改状态。

  5. resolve和reject可以传递参数。

于是我们得到以下代码：

// 先把三种状态声明为常量，方便维护
const PENDING = "pending";
const RESOLVE = "fulfilled";
const REJECT = "rejected";

// 声明一个类，以便我们使用new关键词去创建实例
class LPromise {
  // todoFn就是我们传进来的函数
  constructor(todoFn) {
    // 定义默认参数
    this.status = PENDING;
    // 保存resolve的参数
    this.value = undefined
    // 保存reject的参数
    this.reason = undefined
    // 声明resolve函数和reject函数，以便我们传入的函数调用
    const resolve = (value) => {
        if(this.status === PENDING) {
            this.status = RESOLVE
            this.value = value
        }
    }

    const reject = (reason) => {
        if(this.status === PENDING) {
            this.status = REJECT
            this.reason = reason
        }
    }

    todoFn(resolve,reject)
  }
}

# Promise的执行

我们平时拿Promise结果的时候都会调用Promise的then方法，then方法都做了什么呢？

  1. then方法接收两个参数，第一个参数作为resolve的回调，第二个参数作为reject的回调。

  2. then方法的两个参数是函数，它们各有一个参数，第一个参数的参数是resolve传入的参数，第二个参数的参数是reject传入的参数。

  3. then方法是可以多次调用的（比如声明一个Promise对象，调用两次then方法，会调用两次）。

  4. then方法可以链式调用。

  5. then方法如果**在异步任务（如定时器）**中调用，会在异步任务执行完毕之后触发。

  6. then方法的返回值也是一个Promise

const PENDING = "pending";
const RESOLVE = "fulfilled";
const REJECT = "rejected";

class LPromise {
  constructor(todoFn) {
    this.status = PENDING;
    this.value = undefined
    this.reason = undefined

    /* 
        为了使调用then方法时传入的函数能够在resolve/reject时被执行，我们需要在类中声明一个变量进行存储
        为了支持then方法的多次调用，我们使用数组来储存
    */
    this.resolveFns = []
    this.rejectFns = []
    const resolve = (value) => {
        if(this.status === PENDING) {
            this.status = RESOLVE
            this.value = value

        /*
           当我们执行resolve时，还没有获取到then方法中的参数，如何保证执行resolve时能够保证已经获取到then方法中的参数了呢？ 我们可以将此步骤加入任务队列。
        */
            queueMicrotask(() => {
                this.resolveFns.forEach(fn => fn(this.value))
            })
        }
    }

    const reject = (reason) => {
        if(this.status === PENDING) {
            this.status = REJECT
            this.reason = reason
            // 同resolve
             queueMicrotask(() => {
                this.rejectFns.forEach(fn => fn(this.reason))
            })
        }
    }

    todoFn(resolve,reject)
  }

  then(onFulfilled, onRejected) {
    /* 
        将resolve和reject的回调分别存起来，此时我们需要判断Promise的状态
        1. 如果状态已经确定是fulfilled或者rejected，说明此时resolve/reject已经执行过了，那么resolveFns/rejectFns肯定也就执行过了，我们再往里面添加也不会执行了，那么我们可以直接执行该方法。
        2. 如果状态是pending，说明then方法在同步代码中调用，我们就可以将回调函数存起来，一起调用。
     */
    if(this.status === PENDING) {
        /* 
            我们知道onFulfilled和onRejected是有参数的，参数分别为我们的value和reason变量

            如果我们之间将onFulfilled/onRejected存起来，在resolve/reject中调用时需要遍历resolveFns/onRejected拿到一个个函数，并且将参数传入，与其这样，我们不如多包裹一层函数，只要我们调用这个函数，就可以直接执行onFulfilled/onRejected，这样我们就可以在该步骤将参数传进去。
        */
        this.resolveFns.push(onFulfilled)
        this.rejectFns.push(onRejected)
    }
    if(this.status === RESOLVE) {
         onFulfilled(this.value)
    }
    if(this.status === REJECT) {
        onRejected(this.reason)
    }
  }
}

# Promise.then的返回值

既然上面我们说了then方法的返回值是一个Promise，这样才能支持链式调用，并且无论是resolve还是reject，只要有返回值都会走到下一个then，只有捕获到异常时才会走到下一个catch，因此上面的then方法我们可以改写成如下代码：

 then(onFulfilled, onRejected) {
    return new LPromise((resolve,reject) => {
        if(this.status === PENDING) {
            this.resolveFns.push(onFulfilled)
            this.rejectFns.push(onRejected)
        }
        if(this.status === RESOLVE) {
            try {
                const res = onFulfilled(this.value)
                resolve(res)
            } catch(err) {
                reject(err)
            }
        }
        if(this.status === REJECT) {
             try {
                const res = onRejected(this.reason)
                resolve(res)
            } catch(err) {
                reject(err)
            }
        }
    })
  }

  /* 
    如果这么写的话，pending状态如何处理呢？ 
    如果还是直接将onFulfilled/onRejected直接存起来，我们获取结果就会很麻烦了，我们可以这么做：
    直接在onFulfilled/onRejected外层包裹一层函数，里面做我们的逻辑处理，把最外层的函数存起来，调用最外层的函数，直接执行我们的onFulfilled/onRejected，类似这样：
  */
    if(this.status === PENDING) {
        this.resolveFns.push(() => {
            try {
                const res = onFulfilled(this.value)
                resolve(res)
            }catch(err) {
                reject(err)
            }
        })
        this.rejectFns.push(() => {
            try {
                const res = onRejected(this.value)
                resolve(res)
            }catch(err) {
                reject(err)
            }
        })
    }

    // 如果我们这么做的话，在resolve/reject之中，执行回调函数时，也不需要传参了。

const PENDING = "pending";
const RESOLVE = "fulfilled";
const REJECT = "rejected";

/** 封装一个公共逻辑处理函数
 *  @method 处理then的返回值
 *  @param { function } fn 一个函数，resolve的回调或reject的回调
 *  @param { any } value 回调函数的参数
 *  @param { function } resolve resolve函数
 *  @param { function } reject reject函数
 */
function toCheckTryCatch(fn, value, resolve, reject) {
  try {
    const result = fn(value);
    resolve(result);
  } catch (err) {
    reject(err);
  }
}

class LPromise {
  constructor(todoFn) {
    this.status = PENDING;
    this.value = undefined
    this.reason = undefined
    this.resolveFns = []
    this.rejectFns = []
    const resolve = (value) => {
        if(this.status === PENDING) {
            this.status = RESOLVE
            this.value = value
            queueMicrotask(() => {
                // 这里无需传参处理
                this.resolveFns.forEach(fn => fn())
            })
        }
    }

    const reject = (reason) => {
        if(this.status === PENDING) {
            this.status = REJECT
            this.reason = reason
             queueMicrotask(() => {
                this.rejectFns.forEach(fn => fn())
            })
        }
    }

    todoFn(resolve,reject)
  }

  then(onFulfilled, onRejected) {
   return new LPromise((resolve,reject) => {
        if(this.status === RESOLVE) {
            toCheckTryCatch(onFulfilled,this.value,resolve,reject)
        }
        if(this.status === REJECT) {
             toCheckTryCatch(onRejected,this.reason,resolve,reject)
        }
         if(this.status === PENDING) {
            this.resolveFns.push(() => {
                toCheckTryCatch(onFulfilled,this.value,resolve,reject)
            })
            this.rejectFns.push(() => {
                toCheckTryCatch(onRejected,this.reason,resolve,reject)
            })
        }
    })
  }
}

// 我们发现，如果没有传入onFulfilled或者onRejected怎么办呢？我们来给它一个默认值

// 如果没有传入onFulfilled，我们就让它是一个返回一个undefined的函数
onFulfilled = onFulfilled || ((value) => { return value})

// 如果没有传入onRejected，我们就让它是一个throw异常的函数
onRejected = onRejected || ((err) => { throw err })

class LPromise {
    catch(onRejected) {
        return this.then(undefined, onRejected);

# Promise的finally方法

TIP

Promise的finally其实就是传入一个函数，该函数会在then方法的两个参数中分别执行一遍，这样无论我们执行resolve还是reject就都可以执行finally了。

class LPromise {
   finally(onFinally) {
    return this.then(
      () => {
        onFinally();
      },
      () => {
        onFinally();
      }
    );
  }
}

# Promise的静态方法之all

TIP

  • Promise.all()返回一个Promise
  • Promise.all()需要传入一个Promise的数组，等到数组里的Promise都执行完，把结果放到数组中返回，如果有一个Promise走了reject，那么只返回这个reject
class LPromise {
  static all(promiseArr) {
    // 返回一个Promise
    return new LPromise((resolve, reject) => {
      // 定义一个空数组存放返回值
      const values = [];
      // 遍历这个Promise数组，执行then方法，把结果依次push进去并返回，如果有一个reject，则直接返回reject
      promiseArr.forEach((promise) => {
        promise.then(
          (value) => {
            values.push(value);
            if (values.length === promiseArr.length) {
              resolve(values);
            }
          },
          (reason) => {
            reject(reason);
          }
        );
      });
    });
  }
}

# Promise的静态方法之allSettled

TIP

  • Promise.allSettled()返回一个Promise
  • Promise.allSettled()需要传入一个Promise的数组，等到数组里的Promise都执行完，无论是走resolve还是reject，把结果放到数组中返回。（详情见Promise篇章）
class LPromise {
  static allSettled(promiseArr) {
    // 返回一个Promise
    const result = [];
    return new LPromise((resolve) => {
      promiseArr.forEach((promise) => {
        promise.then(
          (res) => {
            result.push({
              status: "fulfilled",
              value: res,
            });
            if (result.length === promiseArr.length) resolve(result);
          },
          (err) => {
            result.push({
              status: "rejected",
              reason: err,
            });
            if (result.length === promiseArr.length) resolve(result);
          }
        );
      });
    });
  }
}

# Promise的静态方法之race

TIP

class LPromise {

        promise.then(
          (res) => {
            resolve(res);
          },
          (err) => {
            reject(err);
          }
        );
      });
    });
  }
}

# Promise的静态方法之any

TIP

  • Promise.any()返回一个Promise
  • Promise.any()需要传入一个Promise的数组，只要有一个Promise成功，那么就返回成功的结果，如果全部都reject，那么就返回一个失败的Promise。
class LPromise {
   static any(promiseArr) {
    const reasons = [];
    return new LPromise((resolve, reject) => {
      promiseArr.forEach((promise) => {
        promise.then(
          (res) => {
            resolve(res);
          },
          (err) => {
            reasons.push(err);
            if (reasons.length === promiseArr.length) {
              reject(new AggregateError(reasons));
            }
          }
        );
      });
    });
  }
}

# Promise的最终版

const PENDING = "pending";
const RESOLVE = "fulfilled";
const REJECT = "rejected";

function toCheckTryCatch(fn, value, resolve, reject) {
  try {
    const result = fn(value);
    resolve(result);
  } catch (err) {
    reject(err);
  }
}

class LPromise {
  constructor(todoFn) {
    this.status = PENDING;
    this.resolveFns = [];
    this.rejectFns = [];
    this.value = undefined;
    this.reason = undefined;

    const resolve = (value) => {
      if (this.status === PENDING) {
        this.value = value;
        this.status = RESOLVE;
        queueMicrotask(() => {
          this.resolveFns.forEach((fn) => fn());
        });
      }
    };

    const reject = (reason) => {
      if (this.status === PENDING) {
        this.reason = reason;
        this.status = REJECT;
        queueMicrotask(() => {
          this.rejectFns.forEach((fn) => fn());
        });
      }
    };

    try {
      todoFn(resolve, reject);
    } catch (err) {
      reject(err);
    }
  }

  then(onFulfilled, onRejected) {
    onFulfilled =
      onFulfilled ||
      ((value) => {
        return value;
      });
    onRejected =
      onRejected ||
      ((err) => {
        throw err;
      });
    return new LPromise((resolve, reject) => {
      if (this.status === RESOLVE) {
        toCheckTryCatch(onFulfilled, this.value, resolve, reject);
      }
      if (this.status === REJECT) {
        toCheckTryCatch(onRejected, this.reason, resolve, reject);
      }
      if (this.status === PENDING) {
        this.resolveFns.push(() => {
          toCheckTryCatch(onFulfilled, this.value, resolve, reject);
        });
        this.rejectFns.push(() => {
          toCheckTryCatch(onRejected, this.reason, resolve, reject);
        });
      }
    });
  }

  catch(onRejected) {
    return this.then(undefined, onRejected);
  }

  finally(onFinally) {
    return this.then(
      () => {
        onFinally();
      },
      () => {
        onFinally();
      }
    );
  }

  // 静态方法
  static resolve(value) {
    return new LPromise((resolve) => resolve(value));
  }

  static reject(reason) {
    return new LPromise((resolve, reject) => reject(reason));
  }

  static all(promiseArr) {
    return new LPromise((resolve, reject) => {
      const values = [];
      promiseArr.forEach((promise) => {
        promise.then(
          (value) => {
            values.push(value);
            if (values.length === promiseArr.length) {
              resolve(values);
            }
          },
          (reason) => {
            reject(reason);
          }
        );
      });
    });
  }

  static allSettled(promiseArr) {
    const result = [];
    return new LPromise((resolve) => {
      promiseArr.forEach((promise) => {
        promise.then(
          (res) => {
            result.push({
              status: "fulfilled",
              value: res,
            });
            if (result.length === promiseArr.length) resolve(result);
          },
          (err) => {
            result.push({
              status: "rejected",
              reason: err,
            });
            if (result.length === promiseArr.length) resolve(result);
          }
        );
      });
    });
  }

  static race(promiseArr) {
    return new LPromise((resolve, reject) => {
      promiseArr.forEach((promise) => {
        promise.then(
          (res) => {
            resolve(res);
          },
          (err) => {
            reject(err);
          }
        );
      });
    });
  }

  static any(promiseArr) {
    const reasons = [];
    return new LPromise((resolve, reject) => {
      promiseArr.forEach((promise) => {
        promise.then(
          (res) => {
            resolve(res);
          },
          (err) => {
            reasons.push(err);
            if (reasons.length === promiseArr.length) {
              reject(new AggregateError(reasons));
            }
          }
        );
      });
    });
  }
